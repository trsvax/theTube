# Thoughts Skill

How to handle idea exploration and article creation on theTube.

## The trigger

Only draft an article when the user explicitly says **"thoughts"** — either as a prefix (`thoughts: does GitHub solve the mythical man-month?`) or as a clear direct request ("write a post about X").

Any other phrasing — questions, comparisons, "compare this to X", topic mentions — is a **conversation**, not a drafting session. Discuss, don't produce.

## When "thoughts" is triggered

1. Share your take on the topic — a few sentences, not an outline
2. Ask one question to understand what angle the user wants
3. Wait for direction before drafting anything

## When drafting

- Read `skills/post/SKILL.md` before writing
- Match the voice of existing posts: direct, specific, no fluff
- Don't add sections the user didn't ask for
- Short is better than comprehensive — a tight 400 words beats a padded 1,200

## What to never do

- Don't create an article from a topic mention alone
- Don't produce an outline unprompted
- Don't assume a comparison request means "write a comparison post"
